연습 순서는 연결 리스트 -> 스택 및 큐 -> 이진 트리 -> 이진 검색 트리 순으로 진행될 수 있습니다.

각 문제를 푸는 적절한 단계는 다음과 같습니다. 문제지에서 문제 요구 사항을 읽고 -> 해당 메인 프레임을 찾아 C 컴파일러(예: Code::Block)에 복사합니다. -> 함수 부분을 작성합니다. -> 컴파일하고 테스트 케이스를 입력합니다.

각 문제의 메인 프레임은 함수 부분이 비어 있는 상태로 제공되므로, 해당 빈칸을 채워 문제를 완성할 수 있습니다. POP/PUSH/DEQUEUE/ENQUEUE/REMOVE_LINKED_NODE/FIND_LINKED_NODE와 같은 기본 기능은 메인 프레임에 이미 포함되어 있으므로, 이러한 기본 함수는 직접 작성할 필요가 없습니다.

이 문제들은 자료 구조의 기초를 보여주는 예시일 뿐입니다. 하지만 이러한 문제들을 통해 컴퓨터 과학의 새로운 세계로 나아가는 첫걸음을 내딛을 수 있습니다. 이 문제들을 풀고 나면 자료 구조가 무엇인지 간략하게 이해할 수 있을 것입니다.
